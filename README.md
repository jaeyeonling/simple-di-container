# Simple DI Container

[![codecov](https://codecov.io/gh/jaeyeonling/simple-di-container/branch/master/graph/badge.svg)](https://codecov.io/gh/jaeyeonling/simple-di-container)

## 지원 기능
- [x] Inject
- [x] Component
- [x] ComponentScan
